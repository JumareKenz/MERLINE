# Merline

AI-native Monitoring, Evaluation, Research and Learning (MERL) operating system.

## Overview

Merline transforms how organizations design research, collect data, generate evidence, and make decisions. Built for researchers, evaluators, field teams, program managers, and decision-makers across governments, NGOs, development partners, and research institutions.

## Platform

- **Web**: Research management, dashboards, administration
- **Mobile**: Offline-first field data collection
- **AI**: Intelligent research design, data quality, insight generation

## Architecture

- **Backend**: Laravel / PHP / PostgreSQL
- **Frontend**: Next.js / React / TypeScript
- **Mobile**: Flutter / Dart
- **Infrastructure**: Docker / Kubernetes
